# Planisuss
this is the final project for the computer programming class of my AI university course.

It consists in the simulation of a simple ecosystem constituted by a species of plant (Vegetob), a species of erbivore (Erbast) and a species of carnivor (Carviz). Vegetob grows according to soil fertility and animals can form social groups and fight each other.

The world is procedurally generated
